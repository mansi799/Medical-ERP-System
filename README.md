
# Medical ERP System 👑💊

**Built by:** Mansi Rajput, 17 Year Old Backend Developer

## Problem I Solved
Medical stores used books for billing and had no real-time stock tracking.

## My Solution
A 24/7 web ERP with:
- 1-Click Billing + PDF Download
- Auto Stock Deduction 
- Low Stock WhatsApp Alert
- User Login System

## Tech Stack
- **Backend:** Python, Flask
- **Database:** SQLite 
- **Frontend:** HTML, CSS, JavaScript
- **Deployed on:** Real server for 1 medical store
